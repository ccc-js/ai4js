# v0.1: nn4js => ai4js/nn

## 迁移说明

将 `nn4js` 仓库的源代码迁移到 `ai4js/nn` 子目录，作为 ai4js 的神经网络基础模块。

## 变更内容

### 目录结构

```
ai4js/
├── src/nn/                    # 迁入的源代码
│   ├── tensor.js              # 张量类
│   ├── nn.js                  # 网络层与优化器
│   ├── gpt.js                 # GPT 模型
│   ├── chargpt.js             # 字符级 GPT 训练
│   ├── index.js               # 统一导出
│   └── main.js                # 入口
├── tests/                     # 迁入的测试文件
├── ml/                        # 空目录（待扩展）
├── gym/                       # 空目录（待扩展）
├── agent/                     # 空目录（待扩展）
├── data/
│   └── input.txt              # 训练数据
└── test.sh                    # 测试脚本
```

### 文件迁移

| 原始位置 | 新位置 |
|---------|--------|
| nn4js/src/* | ai4js/src/nn/ |
| nn4js/tests/* | ai4js/tests/ |
| nn4js/input.txt | ai4js/data/input.txt |

### 路径修正

- `chargpt_main.js` 中的 `input.txt` 路径从 `../input.txt` 改为 `../../data/input.txt`
- `test.sh` 入口改为 `node src/nn/chargpt_main.js`
