## 已實作

### 1. CLI 訓練工具 (`src/cli/train.ts`)

```bash
# CartPole 訓練
npx tsx src/cli/train.ts --env CartPole --episodes 100 --policy random

# FrozenLake 訓練
npx tsx src/cli/train.ts --env FrozenLake --episodes 100 --policy random

# Q-learning (目前效果不好，連續狀態複雜)
npx tsx src/cli/train.ts --env CartPole --episodes 200 --policy qlearn
```

支援參數：
- `--env` - CartPole 或 FrozenLake
- `--episodes` - 訓練回合數
- `--maxSteps` - 每回合最大步數
- `--policy` - random 或 qlearn
- `--alpha` - Q-learning 學習率
- `--gamma` - 折扣因子
- `--epsilon` - 探索率

### 2. GymReplay 類 (`src/gym/replay.ts`)

```typescript
import { GymReplay, createEpisodeRecord, recordStep } from '../gym/replay.js';

// 創建 episode record
const record = createEpisodeRecord();
recordStep(record, [0, 0, 0, 0], 1, 1, false);
recordStep(record, [0.1, 0.2, 0.01, 0.03], 1, 1, true);

// Replay
const replay = new GymReplay({
  episodeRecord: record,
  onStep: (step, obs, action, reward, done) => {
    console.log(`Step ${step}: obs=${obs}, action=${action}`);
  },
  onDone: (totalReward, success) => {
    console.log(`Done! reward=${totalReward}, success=${success}`);
  },
});

replay.setSpeed(50); // 50ms per step
replay.play();
replay.pause();
replay.stop();
```

### 3. Server API

```
POST   /api/gym              # 儲存 Gym 訓練結果
GET    /api/gym              # 列表 (可 filter by env)
GET    /api/gym/:id          # 取得單個記錄
DELETE /api/gapi/gym/:id     # 刪除
```
