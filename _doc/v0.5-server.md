# v0.4 - Training Server & Real-time Communication

## 目標

- Training 可透過 CLI 先跑完，結果存在 server 端
- Server 透過 REST API (fetch) 將訓練結果傳給瀏覽器
- 即時互動（訓練進度、播放控制）可用 WebSocket
- 視情況選用適當的通訊方式

## 通訊方式選擇

| 情境 | 方式 | 原因 |
|------|------|------|
| 取得訓練歷史 | REST API (fetch) | 一次 request/response |
| 載入特定 training | REST API (fetch) | 隨機讀取 |
| 即時訓練進度 | WebSocket | 伺服器主動推送 |
| 即時播放控制 | WebSocket | 低延遲互動 |
| 刪除/匯出 | REST API (fetch) | 標準 CRUD |

## 架構

```
CLI Train -> Server Storage (JSON files)
                  |
                  +-- REST API (fetch) --> Browser: 取得訓練結果
                  |
                  +-- WebSocket (可選) --> Browser: 即時進度/播放控制
```

## 模組設計

### 1. server/types.ts - 共用類型

```typescript
// ===== 共用類型 =====

interface EpisodeRecord {
  observations: number[][];
  actions: number[];
  rewards: number[];
  done: boolean;
  success?: boolean;
}

interface SavedTraining {
  id: string;
  timestamp: number;
  envName: 'FrozenLake' | 'CartPole';
  episodes: number;
  success: boolean;
  totalReward: number;
  episodeRewards: number[];
  finalRecord: EpisodeRecord;
}

// ===== WebSocket 訊息 =====

type WSType =
  | 'train:progress'
  | 'train:complete'
  | 'train:error'
  | 'replay:step'
  | 'replay:complete';

interface WSMessage {
  type: WSType;
  id: string;
  payload: unknown;
  timestamp: number;
}
```

### 4. server/client.ts - 瀏覽器用戶端

```typescript
class APIClient {
  private baseURL: string;

  constructor(baseURL?: string);

  // REST API
  async listTrainings(): Promise<SavedTraining[]>;

  onComplete(callback: (data: { id: string; success: boolean }) => void): void;

```

### 5. server/trainer.ts - 訓練器

```typescript
interface TrainerOptions {
  envName: 'FrozenLake' | 'CartPole';
  episodes: number;
  maxSteps: number;
  onEpisode?: (episode: number, reward: number) => void;
  onProgress?: (episode: number, reward: number, done: boolean) => void;
}

interface TrainResult {
  id: string;
  success: boolean;
  totalReward: number;
  episodeRewards: number[];
  finalRecord: EpisodeRecord;
}

async function train(options: TrainerOptions): Promise<TrainResult>;
```

### 6. cli/train.ts - CLI 訓練工具

```typescript
// 用法: npx ts-node src/cli/train.ts --env FrozenLake --episodes 100

interface CLIOptions {
  env: 'FrozenLake' | 'CartPole';
  episodes: number;
  maxSteps: number;
  output?: string;  // output directory
}

async function main(options: CLIOptions): Promise<void> {
  const result = await train({
    envName: options.env,
    episodes: options.episodes,
    maxSteps: options.maxSteps,
    onEpisode: (ep, reward) => {
      console.log(`Episode ${ep}: reward = ${reward.toFixed(2)}`);
    },
  });

  const storage = new TrainingStorage(options.output);
  const id = await storage.save(result);

  if (result.success) {
    console.log(`\n✓ Training SUCCESS! ID: ${id}`);
  } else {
    console.log(`\n✗ Training failed after ${options.episodes} episodes`);
  }
}
```

## HTTP API

```
GET    /api/trainings           # 列表 (APIClient.listTrainings)
GET    /api/trainings/:id       # 取得單筆 (APIClient.getTraining)
DELETE /api/trainings/:id       # 刪除 (APIClient.deleteTraining)
POST   /api/trainings           # 匯入 (server 處理)
```

## WebSocket 訊息流程 (可選，用於即時)

### 即時訓練進度 (server → client)

```
Server -> Client: { type: 'train:progress', payload: { episode, reward, done } }
Server -> Client: { type: 'train:complete', payload: { id, success } }
```

### 即時播放 (server → client)

```
Server -> Client: { type: 'replay:step', payload: { step, observation, action, reward, done } }
Server -> Client: { type: 'replay:complete' }
```

## 瀏覽器使用範例

### 取得並顯示歷史訓練

```typescript
import { APIClient } from '../../src/server/client.ts';

const api = new APIClient('http://localhost:3001');

// 取得訓練歷史
const trainings = await api.listTrainings();
console.log('History:', trainings);

// 載入特定訓練結果
const training = await api.getTraining(trainings[0].id);
if (training) {
  showReplay(training.finalRecord);
}
```

### 即時訓練進度 (WebSocket)

```typescript
import { APIClient } from '../../src/server/client.ts';

const api = new APIClient('http://localhost:3001');

// 連線取得即時進度
api.connectWS();
api.onProgress((episode, reward) => {
  document.getElementById('episode').textContent = episode;
  document.getElementById('reward').textContent = reward.toFixed(2);
});
api.onComplete(({ id, success }) => {
  console.log(`Training ${id} complete: ${success}`);
  if (success) {
    api.getTraining(id).then(t => showReplay(t.finalRecord));
  }
});
```
