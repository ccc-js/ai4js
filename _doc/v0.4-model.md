# v0.4 - Save/Load (存檔與載入)

## 目標

先用 chargpt 實驗存檔功能，確認格式可行後再擴展到其他模型。

## 流程

```
chargpt_main.ts (訓練) → model/xxx.json (存檔)
                              ↓
                    (載入) chargpt_main.ts (繼續訓練或生成)
```

### 2. API 設計

```typescript
// src/nn/io.ts

interface ModelSnapshot {
  version: string;
  type: 'nn/gpt' | 'nn/model';
  config: Record<string, unknown>;
  weights: {
    name: string;
    shape: number[];
    data: number[];
  }[];
  metadata: Record<string, unknown>;
}

interface GPTSnapshot extends ModelSnapshot {
  type: 'nn/gpt';
}

// 儲存到檔案
async function saveModel(model: GPT, filepath: string, metadata?: object): Promise<void>;

// 從檔案載入
async function loadModel<T>(filepath: string): Promise<T>;

// JSON string 互轉
function serializeModel(model: GPT, metadata?: object): string;
function deserializeModel<T>(json: string): T;

// localStorage (瀏覽器用)
function saveToStorage(key: string, model: GPT, metadata?: object): void;
function loadFromStorage<T>(key: string): T | null;
```

### 3. 修改 chargpt_main.ts

```typescript
// src/nn/chargpt_main.ts
import { train_model, generate_samples } from './chargpt.js';
import { GPT } from './gpt.js';
import { Adam, setSeed } from './nn.js';
import { saveModel, loadModel } from './io.js';  // NEW
import * as fs from 'fs';
import * as path from 'path';

const MODEL_PATH = path.join(__dirname, '..', '..', 'model', 'chargpt-v1.json');

async function main() {
  const docs = loadDocs();
  const { uchars, BOS, vocab_size, block_size } = analyzeVocab(docs);

  // 嘗試載入已有模型，或建立新的
  let model: GPT;
  if (fs.existsSync(MODEL_PATH)) {
    console.log('Loading existing model...');
    model = loadModel<GPT>(MODEL_PATH);
  } else {
    console.log('Creating new model...');
    model = new GPT(vocab_size, block_size, 1, 16, 4);
  }

  const optimizer = new Adam(model.parameters(), 0.01);

  // 訓練
  console.log('\n--- training ---');
  train_model(model, optimizer, docs, uchars, BOS, block_size, 1000);

  // 存檔
  await saveModel(model, MODEL_PATH, {
    vocab: uchars,
    BOS,
    description: `Trained on ${docs.length} names`,
  });
  console.log(`Model saved to ${MODEL_PATH}`);

  // 生成
  generate_samples(model, uchars, BOS, vocab_size, block_size, 5, 0.5);
}

main().catch(console.error);
```

## GPT 參數取得

GPT 模型的所有權重需能列舉：

```typescript
// gpt.ts 修改
class GPT {
  // 每個模組有 getWeights() / setWeights()
  // parameters() 回傳所有參數
  parameters(): Tensor[] {
    const params: Tensor[] = [];
    for (const p of this.layers) {
      params.push(...p.parameters());
    }
    return params;
  }
}
```
