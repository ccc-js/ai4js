# v0.3: gym - 强化学习环境框架

## 概述

参考 Python OpenAI Gym 框架，设计 JavaScript/TypeScript 版本的强化学习环境接口。包含通用环境基类和两个示例环境：FrozenLake 和 CartPole。

## 目录结构

```
ai4js/gym/
├── index.ts              # 统一导出
├── env.ts                # 环境基类 (Env)
├── discrete.py           # 离散动作空间
├── box.ts                # 连续动作空间
├── spaces.ts             # 空间类型定义
├── frozen_lake.ts        # FrozenLake 环境
└── cartpole.ts           # CartPole 环境
```

## 核心接口

### Space（空间）

```typescript
// 离散空间（有限个离散选项）
class Discrete {
  n: number;                          // 选项数量
  sample(): number;                    // 随机采样
  contains(x: number): boolean;       // 检查是否属于该空间
}

// 连续空间（多维连续向量）
class Box {
  shape: number[];
  low: number[];
  high: number[];
  sample(): number[];
  contains(x: number[]): boolean;
}
```

### Env（环境基类）

```typescript
interface Env {
  observation_space: Space;            // 观测空间
  action_space: Space;                 // 动作空间

  reset(): number[];                   # 重置环境，返回初始观测
  step(action: number): {              # 执行动作
    observation: number[],             # 新观测
    reward: number,                    # 奖励
    done: boolean,                     # 是否结束
    info: Record<string, unknown>      # 附加信息
  };

  render(): void;                      # 可选：渲染环境
  close(): void;                       # 可选：清理资源
}
```

## FrozenLake 环境

### 说明

经典的网格世界强化学习环境。智能体从起点 (S) 出发，通过有限步移动到达目标 (G)。途中需避开陷阱 (H)。

### 8x8 网格地图

```
S F F F F F F F
F H F H H H H F
F F F F H H F F
F F H F F F F F
H F F F H F H F
F F F F F F F F
F H H H F F H F
F F F F F F F G
```

- S: 起点 (0)
- G: 目标 (end)
- F: 冰面 (安全)
- H: 陷阱 (危险)

### 配置

| 参数 | 类型 | 默认值 | 说明 |
|-----|------|-------|------|
| size | number | 8 | 网格大小 |
| max_steps | number | 100 | 最大步数 |
| reward_success | number | 1.0 | 成功奖励 |
| reward_fail | number | 0.0 | 失败奖励 |
| reward_step | number | 0.0 | 每步奖励 |

### 动作空间

`Discrete(4)` - 上、下、左、右

### 观测空间

`Discrete(size * size)` - 当前格子索引

### 状态转移

- 智能体有 probability=1/3 的概率向期望方向的正交方向移动
- 到达 G：done=true, reward=1
- 到达 H：done=true, reward=0
- 超过 max_steps：done=true, reward=0

## CartPole 环境

### 说明

倒立摆问题。智能体通过左右推动购物车，使杆保持直立。

### 物理参数

| 参数 | 值 |
|-----|-----|
| gravity | 9.8 |
| mass_cart | 1.0 |
| mass_pole | 0.1 |
| length | 0.5 (半长) |
| force_mag | 10.0 |

### 配置

| 参数 | 类型 | 默认值 | 说明 |
|-----|------|-------|------|
| max_steps | number | 500 | 最大步数 |
| theta_threshold | number | π/12 (15°) | 角度阈值 |
| x_threshold | number | 2.4 | 位置阈值 |

### 动作空间

`Discrete(2)` - 0: 向左推, 1: 向右推

### 观测空间

`Box([x, x_dot, theta, theta_dot], ...)` - 4维连续空间

| 维度 | 含义 | 范围 |
|-----|------|-----|
| 0 | 小车位置 | [-2.4, 2.4] |
| 1 | 小车速度 | [-∞, ∞] |
| 2 | 杆角度 | [-π/2, π/2] |
| 3 | 杆角速度 | [-∞, ∞] |

### 终止条件

- x > x_threshold 或 x < -x_threshold
- theta > theta_threshold 或 theta < -theta_threshold
- done = true

## 使用示例

### FrozenLake

```typescript
import { FrozenLake } from './gym/frozen_lake.js';

const env = new FrozenLake({ size: 8, max_steps: 100 });
const obs = env.reset();

for (let t = 0; t < 100; t++) {
  const action = Math.floor(Math.random() * 4); // 随机动作
  const { observation, reward, done, info } = env.step(action);
  console.log(`step ${t}: obs=${observation}, reward=${reward}, done=${done}`);

  if (done) {
    console.log(`Episode finished after ${t + 1} steps`);
    break;
  }
}
```

### CartPole

```typescript
import { CartPole } from './gym/cartpole.js';

const env = new CartPole({ max_steps: 500 });
let obs = env.reset();
let total_reward = 0;

for (let t = 0; t < 500; t++) {
  const action = Math.random() > 0.5 ? 1 : 0; // 随机策略
  const result = env.step(action);

  total_reward += result.reward;

  if (result.done) {
    console.log(`Episode finished at step ${t + 1}, total_reward=${total_reward}`);
    break;
  }
}
```

### Q-Learning 示例

```typescript
import { FrozenLake } from './gym/frozen_lake.js';
import { TabularQAgent } from './agent/q_agent.js';

const env = new FrozenLake({ size: 8 });
const n_states = 8 * 8;
const n_actions = 4;

const agent = new TabularQAgent(n_states, n_actions, {
  alpha: 0.1,      // 学习率
  gamma: 0.99,     // 折扣因子
  epsilon: 0.1     // 探索率
});

for (let episode = 0; episode < 1000; episode++) {
  let obs = env.reset();
  let done = false;

  while (!done) {
    const action = agent.act(obs);
    const result = env.step(action);
    agent.learn(obs, action, result.reward, result.observation, result.done);
    obs = result.observation;
    done = result.done;
  }

  if (episode % 100 === 0) {
    console.log(`Episode ${episode}: avg_reward=${agent.getAvgReward()}`);
  }
}
```
